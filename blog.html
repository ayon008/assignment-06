What is Document object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, allowing developers to interact with and manipulate the content, structure, and styles of a webpage using programming languages like JavaScript.

How do you select an element from DOM?

document.getElementById('id')

document.querySelector('.myDiv')

What is event delegation in the context of DOM,and why it is useful?

Event delegation is a technique in JavaScript where you add a single event listener to a parent element to manage events for its child elements. Instead of adding individual event listeners to multiple child elements, the parent element listens for events, and the event is captured as it propagates up the DOM tree.

How do you manipulate an element's attributes and style using the DOM?

element.getAttribute('attributeName');

element.setAttribute('attributeName', 'value');

element.removeAttribute('attributeName');
